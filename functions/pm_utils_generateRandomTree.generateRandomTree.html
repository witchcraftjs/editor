      Experimental

      Generates a random tree structure where the probability of generating children decreases linearly with depth, resulting in a tree that is bushy at the top and sparse towards the maximum depth.

      This function is agnostic to the actual node structure, relying on the caller-provided callbacks to create and link nodes.

      It uses faker.js for randomness (even on the option parameters).

      See generateRandomDoc for an example of how to use it.
